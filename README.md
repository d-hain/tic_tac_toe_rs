# tic_tac_toe_rs

# How to play (Windows)

Go to the [releases](https://github.com/d-hain/tic_tac_toe_rs/releases) and download the latest one.

# How to run

Clone this repo
```shell
git clone https://github.com/d-hain/tic_tac_toe_rs
```

go into the folder
```shell
cd tic_tac_toe_rs
```

and run using cargo
```shell
cargo run --release
```
